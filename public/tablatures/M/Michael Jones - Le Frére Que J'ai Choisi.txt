Le frère que j'ai choisi (2) interprété par Michael Jones

Le frère que j'ai choisi - Michael Jones


Chanson à jouer un 1/2 ton au dessous

L'enchaînement est facile si le F est joué ainsi :
L'index reste au même endroit pour le Am, F et C.
Le majeur et l'annulaire décalent d'une corde vers le haut entre les accords F, C et G.
Ne pas laisser sonner les accords, mais les couper d'un petit silence en étouffant les cordes entre le F et le C puis entre le C et le G pour bien prononcer le rythme.

Am  F  C  G
0   1  0  3
1   1  1  0
2   2  0  0
2   3  2  0
0   0  3  2
0   0  0  3
